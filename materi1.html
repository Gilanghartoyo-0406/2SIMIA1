Contoh penulisan script java script didalam internal html dan external

1.Penggunaan console diketik langsung dijendela terminal langsung 

          
            <!DOCTYPE html> 
            <html lang="en"> 
            <head> 
              <meta charset="UTF-8"> 
              <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
              <title>Document</title> 
            </head> 
            <body>
            </body> 
            </html>

2. Penggunaan console diketik langsung dijendela html dalam blok <script></script> 

          
            <!DOCTYPE html> 
            <html lang="en"> 
            <head> 
              <meta charset="UTF-8"> 
              <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
              <title>Document</title> 
            </head> 
            <body> 
              <script> 
                alert("hallo java script asyik"); 
                console.log("hallo"); 
              </script> 
            </body> 
            </html>

3. Penggunaan console diketik langsung dijendela file external java script (script.js)

          
            <!DOCTYPE html> 
            <html lang="en"> 
            <head> 
              <meta charset="UTF-8"> 
              <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
              <title>Document</title> 
            </head> 
            <body> 
              <script src="script.js"></script> 
            </body> 
            </html>

Isi javascript yg diatas

          
          alert("hallo java script asyik"); 
          console.log("hallo");
